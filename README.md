# Reward Management System

A modern and interactive Reward Management System Dashboard built using Power BI to transform employee recognition into actionable, data-driven insights.

This project focuses on analyzing employee performance, attendance, productivity, badge distribution, and reward allocation through visually rich and interactive dashboards.

---

## Project Overview

Traditional employee recognition systems often rely on assumptions and manual evaluations. This dashboard aims to provide a data-driven approach to employee rewards and recognition by combining performance analytics with interactive visual reporting.

The dashboard enables organizations to:

- Monitor employee performance
- Track attendance and productivity
- Analyze reward and badge distribution
- Identify top-performing employees
- Improve engagement through recognition insights

---

## Tools & Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Data Modeling

---

## Key Insights
- HR and Operations departments received the highest rewards distribution.
- Silver badge category dominated employee recognition.
- Higher attendance generally correlated with improved performance scores.
- Task completion trends helped identify productivity spikes and gaps.
